Generations Of Computer

There are currently five generations of computer :

1st Generation

Duration 1940-1956
Technology Vacuum Tube
Input Punched Cards
Output Electric typewriter
Language Machine Language
Examples IBM 604 , UNIVAC , ENIAC

2nd Generation

Duration 1956-1963
Technology Transistors
Input Magnetic Disk
Output Magnetic Disk Reader (MDR)
Language Assembly Language
Examples UNIVAC 2 , IBM 7030 ,7780

3rd Generation

Duration 1963-1971
Technology IC Chips
Input Keyboard
Output Monitor
Language BASIC,FORTAN
Examples IBM system/360 ,Borroughs 6700

4th Generation

Duration 1971-Present
Technology Microprocessor
Input Keyboard,Mouse,Joystick
Output Monitor,Printer,Speaker
Language HTML,Javascript,Java,CSS,Python
Examples Core i5 ,Core i7

Note:This knowledge or information is described using tables to make it more interesting
